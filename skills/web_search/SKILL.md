# 网页搜索技能
- **用途**：找热点、查资料、搜新闻
- **触发词**：“帮我搜一下”、“查一下”、“找热点”
- **操作**：
  1. 用 general_search 工具搜索关键词
  2. 提取3-5个最相关的结果，附链接
  3. 总结成“哥，这几个信息比较有用：① ② ③”
